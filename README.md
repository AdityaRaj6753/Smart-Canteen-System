# 🍱 Smart Canteen System

A modern, full-stack digital canteen management solution built with the MERN stack. Designed for students to order food seamlessly and for admins to manage orders via QR scanning.

## ✨ Features

### 🎓 Student Side
- **Premium UI:** Glassmorphism-based design with a mesh gradient background.
- **Cart Management:** Add/Remove items and manage quantities with a sleek interface.
- **Secure Payment:** Multiple payment modes (UPI, Cards, Canteen Wallet).
- **QR Order System:** Instant QR code generation after successful payment for order collection.
- **Order Tracking:** View order status and download receipts.

### 🛡️ Admin Side
- **Dashboard:** Real-time analytics and order management.
- **Menu Management:** Add, update, or remove canteen items.
- **QR Scanner:** Integrated scanner to verify and complete student orders.

## 🚀 Tech Stack
- **Frontend:** React.js, Vite, Lucide-React
- **Backend:** Node.js, Express.js
- **Icons:** Lucide React
- **QR Generation:** react-qr-code
