# 🍱 Smart Canteen System (QR-Based)

A full-stack web application designed to eliminate long queues in university canteens.

## ✨ Key Features
- **Secure Admin Authentication:** Login via Gmail OTP system.
- **QR-Based Ordering:** Real-time order placement via QR scanning.
- **Business Insights:** Integrated Chart.js dashboard for revenue and inventory analytics.
- **Inventory Management:** Easy Stock In/Out toggles and automatic quantity tracking.

## 🛠️ Tech Stack
- **Frontend:** React.js, Chart.js, Lucide-React
- **Backend:** Node.js, Express.js
- **Database:** MySQL (XAMPP)
- **Security:** Dotenv, Nodemailer (OTP Service)

## 🔧 Installation
1. Clone the repo: `git clone [URL]`
2. Install dependencies in both folders: `npm install`
3. Setup `.env` file with your credentials.
4. Run Backend: `node index.js`
5. Run Frontend: `npm run dev`
